localStorage is a built-in Web Storage API in JavaScript that lets a website store small amounts of data directly in the user's browser.

Key Characteristics
Persists after browser restarts (unlike sessionStorage).
Stores data as key-value pairs.
Values are stored as strings.
Data is specific to a website's origin (protocol + domain + port).
Typically allows around 5–10 MB of storage per origin (varies by browser).

    localStorage.setItem("username", "Alice");

    const username = localStorage.getItem("username");
    console.log(username); // Alice

    localStorage.removeItem("username");

    localStorage.clear();   // clear everything

- Store an Object:
    const user = {
        name: "Alice",
        age: 25
    };

    localStorage.setItem("user", JSON.stringify(user));

- Retrieve an Object
    const user = JSON.parse(localStorage.getItem("user"));
    console.log(user.name); // Alice

- Listening for Changes Across Tabs
    window.addEventListener("storage", (event) => {
    console.log(event.key);
    console.log(event.oldValue);
    console.log(event.newValue);
    });

    This event fires in other tabs of the same website when localStorage changes.

- Modern Example
    const settings = {
    theme: "dark",
    language: "en"
    };

    localStorage.setItem("settings", JSON.stringify(settings));

    const savedSettings =
    JSON.parse(localStorage.getItem("settings")) || {};

    console.log(savedSettings.theme);
